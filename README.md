## ‚ú® Features

- **Browse Recipes**: Explore a curated collection of recipes from various cuisines
- **Search & Filter**: Find recipes by name, description, or category
- **Recipe Details**: View detailed information including ingredients, instructions, prep time, and difficulty level
- **Modern UI**: Food-oriented design with vibrant colors (red, yellow, green) inspired by fresh ingredients
- **Smooth Animations**: Engaging user experience with smooth transitions and hover effects
- **Category Browsing**: Filter recipes by cuisine type (Italian, Asian, American, Indian, Dessert)

## üöÄ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/h-yzeng/recipe-explorer
cd recipe-explorer
```

### 2. Install Dependencies

```bash
npm install
```

This will install all required dependencies including:

- Vue 3
- Vuetify
- Vue Router
- TypeScript
- Vite
- Other development dependencies

### 3. Run Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (default Vite port)

### 4. Build for Production

```bash
npm run build
```

### 5. Preview Production Build

```bash
npm run preview
```

Preview the production build locally before deployment.
